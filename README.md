# Lunsjplan

Enkel webapp skrevet i HTML, CSS og JavaScript for å gjøre det enklere å planlegge og huske lunsjplaner i løpet av uken.

## Funksjoner

- Se lunsjplan uke for uke
- Detaljer for når ting kan forberedes
- Merk lunsjer etterhvert som de blir laget
- Legg til, rediger og slett uker direkte i appen
- Alt lagres lokalt i nettleseren

## Kom i gang

Klon repoet og åpne `index.html` i nettleseren din. Ingen installasjon nødvendig.

```bash
git clone https://github.com/anbasement/lunsjplan.git
cd lunsjplan
```

## Teknologi

Bygget med HTML, CSS og JavaScript – ingen rammeverk eller avhengigheter.
